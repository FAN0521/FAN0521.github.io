LeetCode:最长回文子串

发表于 2018-07-01 | 分类于 Algorithm |

摘要：给定一个字符串，找到其中最长的回文子串。

Problom：

题目：给定一个字符串，找到其中最长的回文子串。

测试用例：

input output
“babad” “bab” / “aba”
“cbbd” “bb”
“a” “a”
“abc” “a”

Tips: 注意区分回文子串和回文子序列，子序列可以不连续，但子串必须连续。

解法一：动态规划

$ P(i, j) = (P(i+1, j-1)\ and\ S_i == S_j) $

$ P(i, i) = true $

$ P(i, i+1) = (S_i == S_{i+1}) $

先初始化长度为1和2的回文子串，然后依次找长度为3、4、5$\dots$的回文子串，最后输出结果。

Tips: substr(startPos, length)

时间复杂度：$O(n^2)$

空间复杂度：$O(n^2)$

解法二：中心扩展法

枚举每一个中心位置，向前向后搜索字符，找到该位置最长的回文子串。注意区分奇偶数情况。

时间复杂度：$O(n^2)$

空间复杂度：$O(1)$
