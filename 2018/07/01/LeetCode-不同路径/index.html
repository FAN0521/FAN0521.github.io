LeetCode:不同路径

发表于 2018-07-01 | 分类于 Algorithm | | 阅读次数:

摘要：m * n 的方格，从左上角走到右下角，只能向下或者向右走，求不同的路径数。

Problom：

题目：m * n 的方格，从左上角走到右下角，只能向下或者向右走，求不同的路径数。

测试用例：

input output
m = 3, n = 2 3
m = 7, n = 3 28

Tips: 卡特兰数适用于正方形的格子，结果为$C(n, 2n) - C(n-1, 2n)$。长方形的格子应该用动态规划。

解法：动态规划

dp[i][j]表示从左上的（0，0）点走到（i，j）点不同的路径数。

动态方程：要么从上面下来，要么从左边过来，所以 dp[i][j] = dp[i-1][j] + dp[i][j-1]。

初始化：当只有一行或者一列时，也就是当 (i==0 || j==0) 时，只有一种走法。

Tips：可以压缩dp为一维空间，从上往下，每次循环时，dp存储的是上一层的值，因此只要加上dp[j-1]即可。

时间复杂度：$O(n^2)$

空间复杂度：$O(n)$
