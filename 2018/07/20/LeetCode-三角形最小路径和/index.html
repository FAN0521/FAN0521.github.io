LeetCode:三角形最小路径和

发表于 2018-07-20 | 分类于 Algorithm |

摘要：给定一个三角形，找出自顶向下的最小路径和。每一步只能移动到下一行中相邻的结点上。

Problem:

题目：给定一个三角形，找出自顶向下的最小路径和。每一步只能移动到下一行中相邻的结点上。

测试用例：

input output 说明
[[2],[3,4],[6,5,7],[4,1,8,3]] 11 2+3+5+1=11

解法一：动态规划自顶向下

为了节省空间，设置了一个一维dp数组，和一个临时dp数组，用来记录上一层的各个位置的最小和和这一层各个位置的最小和。最后遍历dp数组，找出最小值。

Tips:

*min_element(dp1.begin(),dp1.end())用来找vector中的最小值。

解法二：动态规划自底向上

自底向上更新不需要临时数组去记录，不用处理边界位置，最后更新到第一层只有一个值，可以直接返回，不用找最小值。
