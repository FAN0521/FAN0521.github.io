LeetCode:解码方法

发表于 2018-07-04 | 分类于 Algorithm |

摘要：一条包含字母 A-Z 的消息通过以下方式进行了编码：

‘A’ -> 1
‘B’ -> 2
…
‘Z’ -> 26

给定一个只包含数字的非空字符串，请计算解码方法的总数。

Problem：

题目：一条包含字母 A-Z 的消息通过以下方式进行了编码：

给定一个只包含数字的非空字符串，请计算解码方法的总数。

测试用例：

input output 说明
“12” 2 “AB”(1,2)或者”L”(12)
“226” 3 “BZ”(2,26)或者”VF”(22,6)或者”BBF”(2,2,6)

解法：动态规划

类似于爬楼梯的思想，要么把前一个数字当成一位数，要么把前两个数字当成两位数，当前位置的方案数，等于前一位和前两位的和。

如果当成一位解码，数字不能为0，如果当成两位解码，数字要大于等于10，小于等于26。

可以压缩辅助空间，从$O(n)$到$O(1)$。类似斐波那契数列，可以使用双值迭代的方法进行空间优化。
